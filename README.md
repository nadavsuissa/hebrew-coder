A modern, sleek, and professional platform to learn Python in Hebrew.

## Features
- **Hebrew Interface:** Fully RTL design.
- **Monaco Editor:** Professional code editing experience.
- **Game Visualization:** Visual feedback for code execution.
- **Local Python Runtime:** Powered by Pyodide (runs entirely in your browser).
- **Smart Mentor:** Simple heuristic hints to help students.

## How to Run
1. Navigate to the directory:
   ```bash
   cd hebrew-coder
   ```
2. Install dependencies (if not already installed):
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Technologies
- Next.js 15 (App Router)
- Tailwind CSS
- Zustand (State Management)
- Pyodide (Python WebAssembly)
- Framer Motion (Animations)
